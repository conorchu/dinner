# 環境建置與需求:
Node.js 打造

# 安裝與執行
1.終端機 下載專案
https://github.com/conorchu/dinner.git

2.終端機開啟專案資料夾
輸入 cd dinner

3.安裝 npm 套件
終端機輸入 npm install (package.json 已提供相對應需安裝的套件)

4.安裝 nodemon 套件
終端機輸入 nodemon app.js 或 dev run nodemon

5.開啟程式後，可至 http://localhost:3000 觀看頁面


# 功能
